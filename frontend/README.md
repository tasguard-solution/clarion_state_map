# Clarion Frontend 🎨

The visual interface for the Clarion State Map project. Built with React 19, TypeScript, and Vite, prioritizing performance and premium aesthetics.

## 🚀 Development

### Installation
```bash
npm install
```

### Run Locally
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```

## 🏗️ Architecture

-   **`src/components/hero`**: Contains the core map visualization and associated layout components like `MapView`, `NigeriaMap`, and `NewsCard`.
-   **`src/styles`**: Centralized design system including `colors.css` and global utility classes.
-   **`src/pages`**: Main application views (e.g., `Home.tsx`).

## 🗺️ Interactive Maps

The frontend uses optimized SVG paths generated by the backend. These are rendered as interactive components that support:
-   Hover effects with state highlights.
-   Dynamic scaling based on the viewport.
-   Contextual information panels.

## 🎨 Design System

We use a custom design system focused on:
-   **Premium Typography**: Leveraging modern, clean fonts.
-   **Glassmorphism**: Subtle translucent effects on UI elements.
-   **Dynamic Animations**: Smooth transitions between app states.
